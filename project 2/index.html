Welcome to Bachelor Degree Year 1 in Computer Science

This program introduces foundational concepts in computing, programming, and problem-solving to prepare students for advanced studies in technology.

List of Courses

Introduction to Algorithms

An overview of computer systems, programming basics, and computational thinking. Students learn to write simple programs and understand hardware/software interactions.

Computer Architecture

Focuses on fundamental programming concepts using a language like Python, including variables, loops, functions, and basic algorithms.

Discrete Mathematical Structures

Covers logic, sets, relations, graphs, and trees, essential for understanding algorithms and data structures in computer science.

Fundamentals of Computing and Programming

Introduces programming paradigms, debugging, and basic software engineering principles.

Mathematics for Computer Science

Mathematical foundations including limits, derivatives, and integrals, applied to computational problems.
